## Usage

Download the default branch of a repo from GitHub:

```console
# clone to /repo
$ npx gitget user/repo

# clone to /folder
$ npx gitget user/repo folder

# clone only a specific subfolder to /folder
$ npx gitget user/repo/subfolder folder
```
